# git init
初始化

# git status
觀察

# git add 檔名
將檔案追加到清單
例如: git add .

# git commit -m N
版本更新
例如：git commit -m 1

# git branch
查看分支

# git remote -v
查詢遠端

# git remote add NAME URL 
將本機和Github相連
例如：git remote add ABC

# git remote set-url origin git@github.com:USERNAME/USERREPOSITORT.git
將目錄更改
例如：git remote set-url ABC git@github.com:AAA/BBB.git
